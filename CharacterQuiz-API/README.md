# Character Quiz API
> Written in PHP

## Setup

1. Clone or download this repository into your htdocs/www directory.
2. Run the included .sql files located in ./sql/ on your MySQL Server'
3. Configure the server in ./config/config.php

## Endpoints

- POST /login
- GET /users/
- GET /users/{id}
- POST /users/
- GET /characters/
- GET /characters/{id}
- POST /characters/
- GET /quizzes/
- GET /quizzes/{id}
- POST /quizzes
